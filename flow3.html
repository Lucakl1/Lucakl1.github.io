HTML

HTML eller "Hyper Text Markup Language" er sproget som alle hjemmesider er bygget op omkring, HTML's "job" er at fortælle browseren hvad der findes på siden. I HTML har jeg lært vigtigheden af opbygningen/opsætningen af koden, samt det at kode HTML generelt.

CSS

CSS eller "Cascading Style Sheets" er HTML's bedste ven og virker i praksis ved at "fortælle" hvordan noget skal se ud. Jeg har fået en stor forståelse af CSS, herunder også kunne se at CSS er meget større end som så. Mine færdigheder med CSS er bl.a.:
- Placering
- Responsivt design (se evt under)
- Styling
- Boksmodellen

Responsiv design

Responsive design er ikke en selvfølge på hjemmesider, selvom nogle elementer automatisk skalerer til forskellige skærmstørrelser, er det langt fra alle. Man kan derfor i CSS-koden bruge nogle forskellige strategier, herunder:
- Værdier som skalerer skærmbredde (%, vw)
- Media queries
- Flexbox eller grids

Se "politikker hjemmeside" projektet i browseren

Javascript

Javascript er den underlige fætter. Sproget lægger sig tæt op af HTML og CSS og bruges i samarbejde med dem begge. Det anvendes til at lave specialfunktioner på en webside. Billedet er af Javascript, som bruges til nuværende header i toppen (navigationsbaren). Selvom vi ikke har skulle lære dette sprog endnu, har jeg selv noget erfaring og kan derfor det mest basale i det.
